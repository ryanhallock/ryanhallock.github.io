Projects

OctoDuels - Intergrated Service Project 2022 - Computer Science 1

Concept

The idea for our project was to make a game, we decided to create a game based off `Simon`.

The game was a group project written in Swift under Igis a tool for creating graphical games.

Motivation

The motivation that we could do all these cool ideas and make the game look awesome.

Code Segments

Example 1 - Box Animations

We would create boxes to serve as input relays. Everytime a box was pressed we would set the active state for a duration amount of frames. The way we did this was calling func animate(time: Int) in our Box class.

This allowed us to call it in any box instance simplifying the code process.

Example 2 - Input Handling

Input handling was handled by an array, storing the keymap to input in the game. When a key was pressed the array called playerControls would take in the input and produce a location. The input would then be passed to the correct executor determined by the key press to func handlePlayerKey(executorSide: PlayerSide, key: String).

This allowed us to simplify and allow more than two player if required on the screen, since you can come up with controls to each side.

Example 3 - Color Graphical Hacks

No graphical/game engine is perfect. We had to create some custom functions to deal with some color issues. No String constructor was provided for the Color class.

First, we had to write the colors in a tupple array. The array would store all the hex color codes for player color pallets.

Second, we would pass the tupple array into convertArrayToPallete(colors: [(String, String, String?, String?)]) -> [ActiveColorStyle]. The last two strings are optional as they could be auto generatated, but kept for overrides.

Third, the function would then map all the hex codes to Color class, used in Igis , to a custom function that's name is really long so it will be shortened to func CMIINNADASCFCBDNSWHTRTUT(hexColor: String?) -> Color.

This function would take in the hex color and split and convert it into 3 UInt8 instance's required by the Color class, and then return that new color.

Conclusion
