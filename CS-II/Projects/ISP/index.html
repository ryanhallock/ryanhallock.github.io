Model Scheduler Backend

The ISP I selected is the model scheduler (backend team). The backend of the model scheduler will need to consist of an API good enough to communicate to the frontend, preformant, easy to maintain and easy to understand. The Model Scheduler is used before hand by students to figure out what best class schedule suits them. We use this at Allen to assist students on Arena Scheduling, where we select what time we would like our courses.
